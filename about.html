Informações sobre os dados de Portugal

COVID-19 EM PORTUGAL

Neste mapa foram carregados os casos de COVID-19 por 10.000 habitantes, por concelho. Os dados referem-se ao total de novos casos acumulados até 26/05/2020.
O arquivo em formato GeoJson foi disponibilizado pelo professor.

Informações sobre os dados mundiais

COVID-19 PELO MUNDO

Os dados estão totalizados por quantidade por 100.000 habitantes, por país.
Neste mapa os dados utilados foram trabalhados a partir de uma tabela .csv adquirida do ECDC (ver link em Fontes de Dados abaixo).
Foi gerado um arquivo GeoJson pré-processado no Excel e QGIS. Os dados são referentes ao total acumulado de novos casos até 01/07/2020.
